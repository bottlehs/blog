## MySQL 상태확인, 시작, 중지

```
mysql.server status // 상태확인
mysql.server start // 시작
mysql.server stop // 정지
```

## DataBase

### DataBase 조회

```
show databases;
```

### DataBase 생성

```
create database blog
```

### DataBase 사용설정

```
use blog
```

### DataBase 삭제

```
drop database blog
```
